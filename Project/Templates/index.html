Email Phishing Detector

A machine learning-based API that detects phishing emails by analyzing both the email content and embedded hyperlinks. This project combines text vectorization, feature engineering, and URL risk analyzing to classify emails as either phishing or legitimate. Designed for backend integration, the model outputs predictions in real time based on email body, sender, and hyperlink patterns. Achieved 98% accuracy on email phishing dataset.
